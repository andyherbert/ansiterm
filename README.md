Pipe stdin to an MS-DOS emulated display with ANSI.SYS escape codes enabled.
Also supports ansimusic.

# Usage
In `ansiterm/`:
> cat test.ans | cargo run

# Build
In `ansiterm/`:
> cargo build --release
